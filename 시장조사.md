## 1. 패러다임 시프트: 프롬프트 호출에서 '에이전틱 워크플로우'로
과거의 AI 비용 관리는 사용자가 입력한 1회의 프롬프트(Stateless)에 대한 토큰 비용을 사후에 확인하는 수준이었습니다. 그러나 2025~2026년을 기점으로 다중 LLM이 서로 대화하며 목표를 달성할 때까지 무한 루프를 도는 **자율형 에이전트(Agentic Workflow)** 시대로 진입했습니다.
이에 따라 기존의 전통적인 API 게이트웨이(예: Kong, Apigee)나 클라우드 비용 대시보드로는 통제 불가능한 **'런어웨이 에이전트(무한 폭주)'** 및 **'섀도우 AI'** 현상이 기업의 핵심 재무/보안 리스크로 부상했습니다.

## 2. AI 에이전트 비용 거버넌스 시장 지형 및 핵심 플레이어 (2025~2026)

이 시장은 에이전트의 생애주기와 데이터 흐름을 기준으로 4가지 레이어로 재편되었습니다.

### ① 직접 경쟁: In-path (요청 경로) 예산 강제 및 AI 게이트웨이
API 요청이 LLM에 도달하기 전에 개입하여 라우팅·캐싱·**실시간 킬 스위치**를 작동시키는 레이어.

*   **SatGate (satgate.io):** 에이전트가 외부 API·MCP를 호출할 때마다 트리거되는 '실시간 에스크로' 게이트웨이. 작업 전 예산 할당, 초과 시 강제 중지. 서드파티 유료 API 비용까지 통합 통제. → **가장 직접적인 경쟁자(프리론칭).**
*   **LiteLLM (litellm.ai):** 100+ LLM 규격을 OpenAI 포맷으로 통일한 오픈소스 프록시. VPC 배포 + 가상 키로 팀별 RPM·예산 하드캡. 사실상 표준.
    *   ⚠️ **우리 관점:** 한국은 보안·망분리 때문에 이걸 **무료 자체 호스팅**으로 선회 중 → 한국 세그먼트 최대 경쟁 위협. 한국을 유료 아닌 무료 디자인 파트너로 두는 근거.
*   **Portkey (portkey.ai, Series A $15M) & TrueFoundry:** 엔터프라이즈 AI 제어 평면. Guardrails + 비용 통제 + 예산 소진 시 저가 모델 'Fallback Routing'.

### ② 비용 관측성 도구 (Observability & Tracing)
*   **Langfuse(2026.1 ClickHouse 인수) & Helicone(YC):** 다중 스텝을 트리로 시각화, 스텝별 지연·토큰 경제학 계산. **사후 분석** 특화.

### ③ FinOps / 단위 경제학 통합 플랫폼
*   **CloudZero(Series C) & Vantage(Series A $21M):** AI 토큰 지출을 AWS/GCP 인프라 비용과 결합. "고객사 A 서빙 총원가(COGS)?" 계산용 재무팀 도구.
    *   ⚠️ **우리 관점:** 우리가 "원가 귀속(COGS)"을 전면에 세우면 **이들과 정면 승부.** 차별점은 "실시간·요청 경로"여야 함(이들은 사후·빌링 기반).

### ④ 에이전트 신원 및 자율 결제망 (Agent Identity & Payments)
*   **Skyfire($9.5M) & Payman($13.8M):** 에이전트 자체에 지갑·예산 부여. 유료 페이월 통과, 에이전트 간 마이크로 트랜잭션.

## 3. 주요 LLM / 클라우드 벤더의 한계점과 대기업의 자체 구축(Build) 사례

### 프로바이더 네이티브 기능의 구조적 한계 (Gap Analysis)
*   **OpenAI & Anthropic Admin API:** ✅ 워크스페이스 월 지출 한도는 가능(2025~2026 출시 확인). 하지만 **'상태를 가진 에이전트 루프'를 이해 못 하며**, 연쇄 호출 전체 비용을 사전에 묶는(Escrow) 기능이 없음. → **"왜 OpenAI가 안 하나" 반론 방어의 핵심.**
*   **Azure / AWS / GCP:** 속도 제한·다운그레이드 라우팅은 지원하나 **멀티 벤더(Claude·GPT·Llama 혼용)를 단일 대시보드로 통제 못 함.**

### 시장 내 미충족 수요 (Unmet Needs) — ★ 차별화 후보
1.  **가치 기반 감가(Semantic ROI Kill-switch):** 지금 모든 도구는 "비용"만 봄. $10 써도 유효하면 살리고 $1 써도 헛돌면 차단하는 '문맥 인지형 차단'이 없음. → **COGS보다 방어 가능성 높은 차별화 축(팀 논의 필요).**
2.  **옴니채널 제어 평면:** 카드·암호화폐·DB 접근이 파편화 → CISO의 '글로벌 킬 스위치'가 어려움.

### 대기업 '자체 구축(In-house)' 트렌드 ✅ (검증됨 — 인용 방식 주의)
*   **Uber:** 프로덕션 에이전트 수천 개를 위해 **제로트러스트 아이덴티티 시스템** 자체 구축 — STS가 단기·단일 홉 JWT 발급, SPIRE 연동, MCP Gateway, **<40ms P99**. 비용할당(15h→30min)은 부산물.
    *   🎯 **결정적 인용:** *"Uber가 만들 수 있었던 이유는 기반(SPIRE·기존 인가·자체 추론 인프라)이 이미 있었고, 이를 안전하게 확장할 만큼 이해하는 엔지니어가 있었기 때문."* → **"기반과 엔지니어가 없는 중간 시장 99%는 못 만든다"**는 반론 탄약.
*   🔵 **DoorDash / Shopify / LinkedIn:** 원문이 각 사 엔지니어링 블로그로 인용. 자체 LLM 플랫폼·비용 할당 사례로 언급되나 내 검색에선 Uber만 강하게 확인. 인용 시 출처 명기.

*   **시장 성장:** ✅ AI Gateway $2.4B(2025)→$15B(2035), **CAGR ~20%.** LLMOps CAGR 17~38%(출처별). (원문 "16~21%"는 보수적 하한으로 정확.)

    *   엔터프라이즈: 연 100만$ 이상

## 5. 한국 AI 네이티브 생태계 및 Build vs Buy 전략 (2026)

### 대규모 프로덕션 에이전트 스케일업 *(지출 규모는 인터뷰로 확인 ❓)*
*   **라이너 (LINER):** 글로벌 1,300만 유저, 학술 DB 연동. 검색-추론-요약 자율 파이프라인.
*   **뤼튼 (Wrtn):** MAU 400만+(매출 471억, 2025 공시), B2C+B2B. 다종 모델 로드밸런싱·폴백.
*   **채널코퍼레이션 (ALF):** B2B CS 에이전트를 고객사 쇼핑몰에 이식(1,000개사 도입). 고객사 DB 직접 호출 → 과금 폭탄 방지가 생존 직결.

### 도입 정당화 임계점 (Build vs Buy ROI) ❓ 추론 모델 — 인터뷰 검증 필수
> ⚠️ 아래는 측정된 고객 행동이 아니라 논리 모델. **인터뷰 최우선 검증 대상.**

1.  **월 지출 ₩15M ($10K~$15K) 돌파:** 사내 대시보드로 사용자별/기능별 귀속이 불가능해지는 구간.
2.  **엔지니어링 기회비용 역전:** 시니어 백엔드 인건비 월 800만~1,000만원. 로깅·API 스펙·대시보드 보수에 **0.2 FTE(월 160만~200만원) 이상** 투입 시 → 월 $500~1,500 SaaS가 이득.
3.  **리스크 헤지:** 주말 폭주 사고 한 번($3K~5K)이면 1년 구독료 ROI 즉시 달성.

> 📌 우리 검증 힌트: 실제 스타트업 지출이 예상보다 낮음($500~5K) → 임계점 $10~15K는 **소규모 스타트업 대부분이 아직 못 넘는 선.** 유료 ICP를 스케일업으로 올린 근거와 일치.
