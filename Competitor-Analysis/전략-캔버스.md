# 전략 캔버스 (Blue Ocean Strategy Canvas) — 2026-07 조사

[경쟁사-분석.md](경쟁사-분석.md)의 후속. 경쟁 요소 10개(X축)에 대해 각 플레이어의 제공 수준(Y축, 1~5)을 매겨 가치 곡선을 비교한다.

> ⚠️ 점수는 공식 문서·가격 페이지·서드파티 리뷰 기반의 **추정치**다 (1=없음/매우 낮음, 5=시장 최고 수준). SatGate·Cycles는 프리론칭/v0.1 단계라 "출시된 기능"이 아니라 "공개된 설계·주장" 기준임에 주의. FinOps 점수는 일반적 FinOps 아키텍처 기반 추정(개별 제품 미검증).

## 2. 가치 곡선 읽기

**세 개의 진영이 서로 다른 산에 있다:**

```
높음 5 |  ①실행전통제: SG━LL     ②추적: OB      ⑤재무: FO      ⑧위임/⑨감사: SG
       |
       |        ← 기술 진영 →           ← 관측 →        ← 재무 →
낮음 1 |  OB·FO는 ①에서 바닥    FO는 ⑩에서 바닥   기술 진영은 ⑤에서 바닥
```

- **게이트웨이/enforcement 진영(SG·CY·LL)**: 요소 1·8·9(통제·위임·감사)에서 경쟁하지만 요소 5(재무 언어)는 전부 1~2점
- **관측성 진영(OB)**: 요소 2·6·7·10에서 최고지만 요소 1이 구조적으로 불가능(아키텍처상 경로 밖)
- **FinOps 진영(FO)**: 요소 5에서 유일하게 5점이지만, 요소 1(실행 전) 불가 + 요소 10(스타트업 접근성) 최하

**어느 곡선도 높게 그리지 못하는 조합 = 블루오션 후보:**

> **요소 1(실행 전 통제) × 요소 3(LLM 밖 비용) × 요소 5(유닛 이코노믹스 리포팅) × 요소 10(스타트업 셀프서브)를 동시에 4점 이상으로 가져가는 플레이어가 없다.**

- SatGate가 1·3은 높지만 5는 2점, 10은 미검증(프리론칭) — 그리고 타겟이 엔터프라이즈·플랫폼 운영사라 8·9(위임·감사)에 투자 중
- FinOps가 5는 높지만 1·10이 바닥
- LiteLLM이 1·10은 높지만 3·5가 낮음

이 조합은 정확히 우리 Buyer Persona의 언어와 일치한다: CTO는 "고객 1명당 AI 원가"(요소 3+4+5), 챔피언은 "폭주를 실행 전에 막는다"(요소 1), 둘 다 스타트업 셀프서브(요소 10)로 진입.

## 3. ERRC 초안 (가설 — 인터뷰로 검증)

| | 요소 | 근거 |
|---|---|---|
| **Eliminate** | ⑨ 서명된 감사 증적 (엔터프라이즈 컴플라이언스 수준) | 스타트업 ICP에는 과잉. SatGate가 여기에 투자하는 동안 우리는 생략 |
| **Reduce** | ② 사후 추적 깊이, ⑧ 멀티에이전트 위임 | 추적은 관측성 도구와 경쟁하지 말고 연동(Langfuse export)으로. 위임은 멀티에이전트가 ICP에 보편화될 때까지 보류 |
| **Raise** | ① 실행 전 통제, ③ LLM 밖 비용 커버리지, ⑩ 스타트업 접근성 | 시장 단층선 + 하위 가설 2 + bottom-up 진입 경로 |
| **Create** | ⑤' **스타트업 문맥의 재무 리포팅** — 고객 1명당 AI 원가(COGS), 기능별 마진, 크레딧 소진 카운트다운 | FinOps의 요소 5를 엔터프라이즈 빌링이 아니라 **호출 경로 데이터에서 실시간으로** 만들어내는 것. 어느 진영도 안 하고 있음 |

## 4. 검증 필요 사항 (점수의 약한 고리)

1. **SatGate의 실제 출시 상태** — 점수가 "주장" 기준. 디자인 파트너 단계에서 실제 작동 범위 확인 필요 (GitHub 코드 검증 가능)
2. **FinOps 진영의 AI 기능 실물** — Finout 블로그 링크가 깨져 일반 아키텍처 기반 추정. 개별 제품 데모 확인 필요
3. **요소 5의 실수요** — "유닛 이코노믹스 리포팅"이 진짜 지불 이유인지는 인터뷰 Q9·Q14·Q15로 검증. 프레임워크가 예쁘다고 수요가 있는 건 아니다
4. **요소 7의 무게** — 인라인 프록시 거부감이 실제 도입 거부 사유인지 (챔피언 페르소나 인터뷰 Q13)
