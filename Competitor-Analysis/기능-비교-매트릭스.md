## 읽는 법 — 세 가지 패턴

1. **"전원 ❌" 줄** (굵게 표시): 루프 패턴 감지, 고객당 COGS 실시간 뷰, 크레딧 카운트다운, 출시 전 비용 시뮬레이션, 동종 벤치마크 → moat 후보의 원료. [../Solution/moat-후보.md](../Solution/moat-후보.md)에서 발굴.
2. **"⚠️만 있는" 줄**: 외부 API 비용 추적·차단, 툴 콜 단위 귀속, 예산 위임 → SatGate·Cycles가 주장하지만 아무도 출시 못 한 영역. 속도 싸움.
3. **"✅ 많은" 줄**: LLM 로깅, 팀별 대시보드, 세션 예산 → 이미 상품화(commoditized). 여기서 차별화 불가, 기본기로만 필요.

> ⚠️ 한계: ❌는 "2026-07 조사에서 확인 안 됨"이지 "영원히 없음"이 아니다. 특히 SG·CY의 ⚠️는 출시되면 ✅로 바뀐다. 분기별 재조사 권장.
