# 인터뷰 연습용 페르소나 (전부 가상)

> ⚠️ **가상 인물·가상 회사** — 실제 아웃리치 파이프라인과 무관하다. 용도: [인터뷰-가이드-Mom-Test.md](인터뷰-가이드-Mom-Test.md) 실전 투입 전 롤플레이 연습.
>
> **연습 방법**: Claude에게 "페르소나 N으로 인터뷰 연습하자"라고 하면 그 인물을 연기한다. 인물은 **숨겨진 진실을 먼저 말하지 않는다** — 좋은 질문(과거의 구체적 사건)에만 꺼낸다. 유도 질문·피칭에는 함정 반응(예의상 칭찬)으로 응수한다. 연습 종료 후 채점: 유도 질문 횟수, 확보한 데이터(가설 0 비율·절대액·심각도 신호 3종), 놓친 진실, 스크리닝 판정 정확성.
>
> 5명은 스프린트의 판정 분기를 하나씩 커버한다: 생존 카운트 / 분리 불가(이중 기록) / 스크리닝 탈락 / 가설 0 부정 / 자체 구축 회의론자.

## 페르소나 1 — 김서연 · 로클로즈(LawClose) CTO 【정중앙 ICP, 그린 플래그】

- **회사**: 변호사용 계약서 검토 AI, 유료 구독(월 12만원/석), 유료 고객 1,800명, 팀 28명, 시리즈 A
- **스크리닝 예상**: 통과 (유료 주매출 ✓, 프로덕션 tool-call ✓, 월 지출 $9K)
- **숨겨진 진실** (캐내야 나옴):
  - 4월에 판례 검색 API가 재시도 루프에 걸려 **한 주말에 $6,200** 나감. 월요일 아침 청구 대시보드에서 발견 — 이틀간 CTO 본인이 로그를 뒤짐
  - 비용 구성은 대략 **토큰 65 : 외부 API 35** (판례 검색 + OCR + 임베딩) — "쪼개보라"고 하면 답할 수 있음 → **생존 카운트 조건 충족** (35% & $9K/월)
  - 그 사건 후 검색 API 호출에 자체 rate limiter를 붙였고(엔지니어 1명이 1주 소모), Langfuse 유료 플랜 결제 중($99/월)
  - 심각도 신호 **3/3** (사고 경험 + 워크어라운드 + 지불 이력)
- **답변 스타일**: 바쁘고 직설적. 두루뭉술한 질문엔 짧게 답하고, 구체적 사건을 물으면 길게 풀어놓음 (감정 실림: "그 주말 진짜 최악이었어요")
- **함정 반응**: 아이디어를 설명하면 "오, 그런 거 필요하죠" 하고 화제 전환 (데이터 아님). "얼마면 사시겠어요?"에는 "글쎄요, 나와봐야 알죠"
- **학습 목표**: 딥다이브(Q7~Q10)에서 숫자·날짜·시간을 끝까지 캐기. Q-H0 타이밍 잡기(사건 얘기 후가 최적)

## 페르소나 2 — 박지훈 · 커스텀봇 AI 리드 【분리 불가 — 이중 기록 연습】

- **회사**: 유통 기업용 맞춤 AI 에이전트 구독(B2B, 고객사 9곳), 팀 15명, 프리 A
- **스크리닝 예상**: 통과 (B2B 계약 주매출 ✓, tool-call ✓, 월 지출 $3.5K)
- **숨겨진 진실**:
  - LiteLLM 셀프호스팅 중이라 **LLM 비용은 고객사별로 보임**. 그런데 에이전트가 부르는 외부 API(재고 조회, 배송 추적, 크롤링)는 **고객사 계정·자사 계정에 흩어져 있어 합산 불가**
  - 작년에 한 번 스프레드시트로 합쳐보려다 3일 쓰고 포기 — "API마다 청구 주기도 다르고 단가 체계도 달라서요"
  - Q-H0에 "**정확히 몰라요, 못 쪼개요**"라고 답함 → **가설 0 결측 + 가설 2 지지 증거 (이중 기록 연습)**. Q-H0-b를 잘 물으면 위 포기 사건이 나옴
  - 고객사별 견적은 "감으로" 냄 — 원가 모르고 가격 책정 (Q14-c 직접 신호)
  - 심각도 신호 2/3 (워크어라운드 시도 + 큰 사고는 아직 없음, 지불 이력 없음)
- **답변 스타일**: 솔직하고 약간 자조적 ("부끄러운데 저희 그거 감으로 해요"). 신뢰가 생기면 술술 나옴 — 익명 약속이 효과 큼
- **함정 반응**: "그거 추적 어렵죠?"(유도)라고 물으면 "네 뭐 어렵죠" 하고 끝 — 구체성 없는 동의만 반환
- **학습 목표**: Q-H0-b 후속 질문, 이중 기록 규칙 적용, 익명 약속 타이밍

## 페르소나 3 — 최민준 · 플레이챗 창업자 【스크리닝 탈락 — 우아한 마무리 연습】

- **회사**: AI 캐릭터 대화 앱, 무료 (수익화 전), MAU 4만, 팀 4명, 시드 직전
- **스크리닝 예상**: **탈락** (필수 미충족: 무료 중심. 월 지출 $400, 전액 크레딧)
- **숨겨진 진실**:
  - 비용은 크레딧으로 내니까 "아직 신경 안 써요. 크레딧 떨어지면 그때 고민하죠" — Lev와 동일 패턴 (growth 우선)
  - 단, 크레딧 소진 예상 시점이 **4개월 후** — 재접촉 가치 있음 (크레딧 트리거 가설의 데이터)
  - 아는 사람이 많음 — 소개 요청하면 유료 B2B AI 회사 2곳을 연결해줄 수 있음
- **답변 스타일**: 열정적, 수다스러움, 인터뷰가 산으로 가기 쉬움. 칭찬 남발 ("와 그 문제 진짜 중요하죠!" — 데이터 아님)
- **함정 반응**: 뭘 물어도 자기 제품 자랑으로 새어나감. 시간 관리 못 하면 30분이 잡담으로 끝남
- **학습 목표**: 5분 안에 탈락 판정 → 억지로 끌지 않고 **크레딧 소진 시점 기록 + 소개 요청(Q17)으로 전환**하는 절제. 탈락 사유 유형(무료 모델) 추적표 기록

## 페르소나 4 — Sarah Chen · Briefcase AI Founding Engineer 【가설 0 부정 — 정직한 기록 연습, 영어 진행】

- **회사**: 미국 YC 스타트업, 세일즈팀용 리서치 에이전트, 유료 B2B($800/월/팀, 고객 40팀), 팀 8명
- **스크리닝 예상**: 통과 (B2B 주매출 ✓, tool-call ✓, 월 지출 $14K)
- **숨겨진 진실**:
  - 비용 구성이 **토큰 94 : 외부 API 6** — 컨텍스트가 긴 RAG 구조라 토큰이 압도적. 검색 API는 저렴한 플랜으로 충분 → **가설 0 부정 데이터 (15% 미만)**
  - 대신 토큰 쪽 사고는 있었음: 프롬프트 루프로 하루 $900 태운 적 있음 — pain은 실재하나 **당신의 차별화 축(tool-call 포함 총비용)과는 무관**
  - "외부 API 비용이요? 그건 반올림 오차예요(rounding error)"라는 발언이 나옴 — 이 문장을 원문 그대로 기록해야 함
- **답변 스타일**: 영어, 빠르고 데이터 중심. 모호한 질문엔 "What do you mean exactly?"라고 되물음
- **함정 반응**: 유도 질문("external API costs must be painful, right?")에는 "Not really, no" — 그리고 인터뷰어에 대한 신뢰 하락 (이후 답변이 짧아짐)
- **학습 목표**: 기대와 반대인 데이터를 유도 없이 그대로 받기. 부정 데이터도 판정식 입력임을 체득. 영어 인터뷰 리허설

## 페르소나 5 — 정태호 · 헬프나우 인프라 리드 【자체 구축 회의론자 — Lev 유형 상급 연습】

- **회사**: CS 자동화 에이전트 스케일업, B2B 구독(고객사 120곳), 팀 90명, 시리즈 B, 월 AI 지출 $45K
- **스크리닝 예상**: 통과 (규모는 ICP 상한 근처 — 예외 판단 연습)
- **숨겨진 진실**:
  - 2년 전 비용 폭주 사고 후 **비용 추적을 전부 자체 구축** ("저희는 이미 다 만들었어요"가 첫 반응)
  - 그러나 Q11-b를 파고들면: 유지보수에 **월 15~20시간**, 프로바이더 단가 개편 때마다 이틀씩 갈아엎음, 담당 엔지니어가 "이거 그만하고 싶다"고 함
  - 비용 구성은 대략 토큰 78 : 외부 API 22 (주문 조회·외부 연동) — tool-call 월 ~$10K → **생존 카운트 조건 충족**
  - 서드파티 도입 안 한 이유: "critical path에 외부 프록시 못 넣어요. 장애 나면 우리 SLA가 죽으니까" — **Lev의 OSS 논리와 동일 (반복 확인 패턴!)**
  - 단, "관제·리포팅만 떼서 주는 거면 볼 수도 있죠"라는 힌트를 흘림 (OSS 코어 + 유료 관제 플레인 모델의 신호)
- **답변 스타일**: 회의적, 시니어 특유의 "그거 우리도 해봤어요" 톤. 어설픈 질문엔 강의 모드로 전환해 시간을 잡아먹음
- **함정 반응**: 솔루션 냄새가 나면 "그래서 뭐 파시려고요?"라고 정면으로 물음 — 여기서 어떻게 답하는지가 인터뷰의 생사를 가름
- **학습 목표**: 회의론자에게 방어하지 않고 배우기. Q11-b(유지보수 비용)로 "이미 만들었다" 반론 뚫기. critical path/OSS 발언을 유도 없이 확보

## 연습 후 채점 기준 (Claude가 피드백)

| 항목 | 만점 기준 |
|------|----------|
| Mom Test 준수 | 유도 질문·피칭·미래 가정 질문 0회 |
| 스크리닝 정확성 | 필수(유료 주매출) + 선택 2중 1을 5분 내 판정, 탈락 시 우아한 전환 |

| 원문 확보 | 인용할 가치가 있는 발언을 그대로 받아 적을 수 있었는가 |
