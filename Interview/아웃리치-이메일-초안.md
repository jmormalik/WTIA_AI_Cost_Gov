# 인터뷰 요청 이메일 초안 (회사별 맞춤)

> 사용법: `[ ]` 부분은 보내기 전에 직접 채우거나 확인. 훅으로 쓴 회사별 사실은 2026-07 조사 기준이니 발송 전 최신 여부 확인 권장. 수신자는 CTO/AI 리드 (링크드인·회사 블로그 저자에서 특정).
>
> 원칙 (Mom Test): 솔루션 피칭 없음 — "만들고 있는 것"이 아니라 "배우고 싶은 것"을 말한다. 20~30분, 상대 경험이 주인공, 부담 없는 거절 여지.
>
> ⚠️ **가설 0 문항(토큰 vs 외부 API 비용 쪼개기)은 이메일에 넣지 않는다** — 설계 결정(2026-07-23): COGS 구성은 가장 민감한 숫자라 인터뷰에서 라포 형성 후(가이드 후반부)에만 묻는다. 이메일의 역할은 그 대화가 자연스럽도록 **영역만 프레이밍**하는 것: "토큰뿐 아니라 에이전트가 호출하는 외부 API까지 합친 총비용" (영역 공개 = Mom Test 허용, 가설 제시 금지).
> 신뢰 장치: 모든 대화 내용은 **익명으로만 정리**한다는 약속을 이메일에 한 줄 포함 (기밀 규칙 — 회사·인물 식별 정보와 구체 숫자는 외부 공유 시 익명화·범위화).

## 공통 뼈대

1. 나는 누구 (1줄) — WTIA 국제 창업 교육 과정, 아이디어 검증 단계
2. 왜 하필 당신 회사인가 (회사별 훅 — 숙제했다는 증거)
3. 무엇을 배우고 싶은가 (문제 영역만, 솔루션 없음)
4. 요청 (20~30분, 화상/대면, 일정 유연)
5. 부담 제거 (거절 여지, 결과 공유 제안)
6. 익명 약속 (대화 내용은 익명으로만 정리 — 결과 공유 제안 문장에 자연스럽게 붙인다)


**v2 템플릿 (2026-07-23 — 총비용 프레이밍 + 익명 약속 반영, 고정부 ~264자, 훅 예산 ~35자):**

- 신뢰 장치는 실명 소속(WTIA) + 익명 약속 두 겹 — 민감한 비용 이야기를 꺼낼 상대라 익명 약속이 특히 효과적
- 비동기 우선 ("메시지로 몇 가지") — 콜 요청은 pain 신호가 확인된 뒤에 (Lev 검증 공식과 동일)
- "no pain" 답장 대응은 섹션 6의 공식 재사용: 인정 → 이유 질문(단일 호출/저볼륨/자체 해결 3지선다) → 소개 요청
- 응답이 오면 즉시 [아웃리치-파이프라인.md](아웃리치-파이프라인.md)에 상태 기록

## 발송 운영 규칙

- **수신자 선정 규칙: 회사가 작을수록 CTO에게, 클수록 실무 리드에게.** 팀 10~30명이면 CTO(본인이 챔피언), 50명 이상이면 AI/ML/인프라 리드. 특정 순서: 기술 블로그 저자(최우선 — 글이 곧 훅) → 링크드인 검색 → 컨퍼런스 발표자 → GitHub 조직 멤버. 같은 회사 두 명에게 동시 발송 금지 — 무응답 시 후속 1회 후 다른 사람으로. 이메일을 못 찾으면 링크드인 DM도 가능
- **한 번에 한 통씩 개인화 확인** 후 발송 — 회사명만 바꾼 티가 나면 응답률이 급락한다
- **한국 트랙 채널은 링크드인에 한정하지 않는다** (설계 결정 2026-07-23) — 이메일·회사 문의채널·커뮤니티·소개를 동순위로. 후보 리스트가 이미 "채널톡 CTO 링크드인 검색 안 됨" 등을 기록해뒀다. 수신자를 못 찾으면: 기술 블로그 저자 → 링크드인 검색 → 공식 문의 채널 → 소개 요청 순
- **후속(follow-up)은 1회만**, 4~5일 후 짧게 ("혹시 놓치셨을까 해서요. 어려우시면 편하게 무시하셔도 됩니다.")
- **모든 발송·응답·상태 변화는 [아웃리치-파이프라인.md](아웃리치-파이프라인.md)에 기록** (접촉 1명 = 1행, 주 1회 이상 갱신 — 스프린트의 모든 정량 트리거가 이 표에서 읽힌다). 후보 리스트에는 회사 단위 결과만 반영
- 응답이 오면 **ICP 스크리닝 (필수: 매출의 주된 원천이 유료 구독/B2B + 선택 2개 중 1개: 프로덕션 자율 tool-call / 월 AI 지출 $500+)** 통과 여부 판정 후 인터뷰 진행. 뤼튼 유형(무료 B2C)은 보조 표본 — 명시적 예외로만, 카운트 제외
- 인터뷰 성사 시 [인터뷰-가이드-Mom-Test.md](인터뷰-가이드-Mom-Test.md)로 진행, 기록은 `YYYY-MM-DD-대상.md`. **가설 0 문항은 필수 확보 항목이되 라포 형성 후(가이드 후반부)에** — 질문 직전에 익명화 약속을 다시 전달. 인터뷰가 짧아지면 남은 시간 10분 시점에 우선 질문
